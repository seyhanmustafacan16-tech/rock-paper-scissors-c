# Rock Paper Scissors in C

A console-based Rock Paper Scissors game developed in C where the player competes against the computer until the selected target score is reached.

## Features

- Play against the computer
- Random computer move generation
- Custom target score selection (1–100)
- Real-time score tracking
- Input validation for both score and player moves
- Round-by-round game results
- Final winner announcement

## Technologies

- C
- Standard C Library (`stdio.h`, `stdlib.h`, `time.h`)

## How to Run

1. Compile the `main.c` file using any C compiler.
2. Run the generated executable.
3. Enter the target score (between 1 and 100).
4. Choose one of the following moves:
   - **T** → Rock
   - **K** → Paper
   - **M** → Scissors
5. Continue playing until either you or the computer reaches the selected target score.

## Project Purpose

This project was developed as part of my C programming learning process to practice fundamental programming concepts, including functions, loops, conditional statements, arrays, random number generation, and user input validation.
