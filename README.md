# Rock Paper Scissors in C

A console-based Rock Paper Scissors game developed in C.

## Features

- Play against the computer
- Random computer moves
- Target score selection
- Score tracking
- Input validation

## Technologies

- C
- Standard C Library

## How to Run

1. Compile the `main.c` file using any C compiler.
2. Run the generated executable.
3. Choose your target score.
4. Play against the computer until one side reaches the target score.

## Project Purpose

This project was developed as part of my C programming learning process to practice fundamental programming concepts such as functions, loops, conditional statements, random number generation, and user input handling.
